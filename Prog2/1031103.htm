Problem 1031103 : ERO on Rationals


Problem Description

You are to write a program that performs the three elementary row operations on an inputted matrix of Rational objects. The program should loop indefinitely until an exit command is entered.

By default, the matrix size is 0 by 0.

The program may only process the following commands:

size r c resizes the matrix to r rows and c columns
exit exits from the program
er1 k r multiply all values in row r by k
er2 k a b subtract from each number in row a the corresponding value in row b times k
er3 a b swaps the corresponding values in row a and row b
data populate the matrix with values read row by row (starting from row 0) from standard input


NOTE: In this problem, you are REQUIRED revise the Matrix class we discussed. You may either transform the Matrix class as a generic Matrix class or simply rewrite the Matrix class so that the data in it are Rational objects. Otherwise, your solution will be marked wrong.

Input Format

The input consists of commands and parameters.

Output Format

For every input command, output the current contents of the matrix. The program should always output the contents of the matrix until the command entered is exit.

Sample Input

size 2 3 
data
1 -3 11
2 1 1
er2 2 1 0
er1 1/7 1
er2 -3 0 1
data
3 4 2
1 2 3
er3 0 1
er2 3 1 0
er2 -1 0 1
er1 -1/2 1
exit
