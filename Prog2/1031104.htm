Problem 1031104 : Arithmetic on Rationals


Problem Description

You are to write a program that performs the four arithmetic operations on Rational objects.

Input Format

The input starts with an integer N > 0, followed by N input statements. Each input statement starts with a Rational value followed by an operation ('+','-','*','/') and followed by another Rational value

Output Format

For every input statement, print the result of the operation in lowest form

Sample Input

3
1/2 + 1/3 
-5 * 1/3
2/5 / 2

Sample Output

5/6
-5/3
1/5

Sample Run

3
1/2 + 1/3
5/6
-5 * 1/3
-5/3
2/5 / 2
1/5
